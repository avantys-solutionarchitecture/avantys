Currently running services

Avantys currently has a total of 6 running microservices that are all used to run Avantys' infrastructure. This (currently static) page allows you to easily navigate to the different services.

Each service is responsible for its own API and documentation, if you have any issues or experience bugs please feel free to notify us through social media.

Please note that Avantys is not a real organisation and is merely a school project.
